## Getting inputs:
Don't forget to add/update your **.secrets** then run:
```sh
./setup.sh [YEAR] [DAY]
```

## formatting

### python

  ```sh
  autopep8 --in-place dayx.py
  ```

### ruby
  Install [rufo](https://github.com/ruby-formatter/rufo)

  ```sh
  gem install rufo
  ```

  and run:

  ```sh
  rufo .
  ```
